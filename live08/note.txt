#### SQLite3 ####

- Uma biblioteca em linguagem C que implementa um banco de dados SQL embutido

- Programas que usam a biblioteca SQLite podem ter acesso a banco de dados
SQL sem executar um processo SGBD(sistema gerador banco de dados) separado.

- Não é uma biblioteca cliente usada para conectar com um grande servidor  de
banco de dados, mas sim o próprio servidor.

- A biblioteca SQLite lê e escreve diretamente no arquivo de banco de dados
no disco.
